Nuxt.js

# nuxt 설치 방법
yarn add nuxt@1.4.1
또는
npm install --save nuxt@1.4.1

# nuxt 간단한 명령어
nuxt : 개발 버전으로 실행 ( 이때 핫리로딩 기능 지원)
nuxt build : 애플리케이션 빌드
nuxt start : 운영 버전으로 서버를 시작 (nuxt build를 먼저 실행)
nuxt generate : 애플리케이션을 빌드한 후 모든 경로의 화면을 정적 HTML로 생성
